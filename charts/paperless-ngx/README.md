## Storage Configuration

### Storage Classes
Each persistence volume (data, media, consume) can be configured with a specific storage class:

```yaml
persistence:
  data:
    storageClass: "fast-ssd"  # Use specific storage class
  media:
    storageClass: ""          # Use default storage class
```

To see available storage classes in your cluster:
```bash
kubectl get storageclass
```

### PVC Management
The chart supports both automatic PVC creation and using existing PVCs:

**Automatic PVC creation (default):**
```yaml
persistence:
  data:
    enabled: true
    size: 10Gi
    storageClass: "standard"
```

**Using existing PVC:**
```yaml
persistence:
  data:
    enabled: true
    useExistingPvc: "my-existing-pvc"
```
